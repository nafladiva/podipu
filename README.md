# Podipu

A Podcast Mobile App, where you can listen to a variety of the best podcasts, explore podcasts by genres, save podcasts you want to listen to later, and much more!âœ¨

## Tech stacks
- Flutter 3.10.5
- Bloc
- Hive

### Important
If you want to run the app, please generate your own ListenNotes's API Key in https://www.listennotes.com/api/ , and place the key in ``` lib/common/secrets/my_secret.dart ```
